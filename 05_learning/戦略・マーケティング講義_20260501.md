## セッション1：戦略的思考の極意
**テーマ：大手の仕組みを無効化し、ジャンプする技術**

### 核心メッセージ
- 「戦略」の本来の意味は「戦いを略す（やらないことを決める）」こと
- 誠実な努力の延長線上だけでは「決定的なジャンプ」は生まれない
- 同じ土俵で戦うこと自体が、最大の「非戦略」

### 大手の仕組みという残酷な構造
市場のルールは「大手が勝つように」設計されている。
- 規模の経済（圧倒的な量によるコスト削減）
- 巨大な資金力（リソースによる持久戦）
- 効率の最大化（全体最適化されたオペレーション）

### 枠組みからジャンプするための2つの思考法
| 思考法 | 問い | 目的 |
|--------|------|------|
| クリティカルシンキング（批判的思考） | 「本当にこの市場ルールは絶対か？」「この前提は誰にとって都合が良いのか？」 | 常識を疑い、構造の歪みを見抜く |
| ゼロベース思考（白紙思考） | 「制約が一切ないとしたら、本来どうあるべきか？」 | 既存の仕組みやアセットへの固執を捨て、発想をジャンプさせる |

### 大手を解剖するスキャナー：フェルミ推定の応用
フェルミ推定 = 対象の「解像度を極限まで上げる」構造分解ツール
1. 前提確認：何を数えるか（収益構造）を明確に定義
2. 戦略策定：需要側・供給側どちらのアプローチか決める
3. 論点の構造化：影響力の大きな切り口でブロック状に因数分解

**目的：大手が「全体最適」のために何を切り捨てているかを特定する**

### 大手の「死角」を見つける
大手の弱点 ＝ 「効率化の裏側」に隠れている。  
大手がマス（多数派）を取りにいくために必ず切り捨てる要素：
- **個別対応（パーソナライズ）**
- **手間のかかるニッチ市場**
- **非合理だが熱狂的な感情**

### 戦略の核：MoSCoW法
| 分類 | 意味 |
|------|------|
| Must Have | 絶対に必要 |
| Should Have | 重要だが後回し可 |
| Could Have | あれば良い |
| **Won't Have** | **今回絶対にやらないこと** ← ここが戦略の核 |

「Won't」を明確に言語化する勇気が、限られたリソースを大手の「死角」へ集中投下する圧倒的な力を生み出す。

### 論点設計（イシュー特定）の重要性
- 論点が間違っていれば、いかに優れた解決策もすべて無駄
- イシューツリーで「なぜ？」を繰り返し、真のボトルネックを特定
- 表面的な課題（売上がない）ではなく、解くべき真の問い（顧客が本当に望む体験が提供できていないのはなぜか？）を設定する

### 優先順位付け：RICEスコア
```
(Reach × Impact × Confidence) ÷ Effort = RICEスコア
```
- Effort が少なく Impact が絶大なものから着手
- 「思い込み」や「政治的判断」を排除し、データで冷徹に優先順位を叩き出す

### ジャンプした企業たちの共通点
| 企業 | 大手のルール（前提） | 見つけた真のイシュー | 捨てたもの（Won't） |
|------|---------------------|---------------------|---------------------|
| CADDi | 相見積もりは営業の勘 | 図面データの価値化 | 自社での製造機能 |
| JINS | 眼鏡は医療器具・高価 | 手軽なファッション化 | 複雑な価格オプション |
| MonkeyFlip | 万人受け・マス狙い | 強烈な自己表現の具現化 | 「誰にでも似合う」デザイン |

**共通法則：大手のルール（効率や常識）には絶対に乗らない。自ら設定した「新たな論点」でゲームを再定義し、捨てる決断をしている。**

### 虎谷事例：DECORTE美容液キャンペーン
- WEBではなくオフライン＋地上波中心に広告投資（社内ではめちゃくちゃ怒られた）
- 日本のメディアの収益構造から仮説を立て有識者ヒアリングで決断
- アンバサダービジネスの構造上の穴を突いた結果
- 投資額の6〜7倍の広告効果、期間中売上は前年の**2倍**、その年の社長賞受賞

### 明日から実践できるアクション
> **会議の前に「最大の論点は何か？」を1行だけメモする**  
> 議論が枝葉に逸れそうになった時、真のボトルネックへ引き戻すコンパスになる。

## セッション2：競争戦略としてのマーケティング
**テーマ：選ばれる「独自の意味」を設計する**

### マーケティングの進化
- **過去**：売れる仕組みをつくること
- **現在**：価値の創造と浸透（より上位概念を包括）

### sirokの現在地：「最強のエンジン」
D2C領域における強力な原動力として：
- 緻密なデータ分析とCPAの最適化
- 高速なPDCAサイクルと仮説検証
- デジタルチャネルにおける卓越した集客力

**→ これは決して否定されるべきものではない。しかし、エンジンだけでは乗り越えられない壁が近づいている。**

### 飽和市場の罠：機能と広告量の競争は「死」を招く
コモディティ化が進む市場で起きること：
- 機能の模倣（スペック競争）
- 広告費の高騰（CPAの悪化）
- → 利益率の圧迫・価格競争による「レッドオーシャン」

機能が優れていること、広告を多く打つことだけでは「選ばれ続ける理由」にならない。

### 競争のパラダイムシフト：「機能」から「意味」へ
| これまでの競争（機能訴求） | これからの競争戦略（意味の設計） |
|--------------------------|-------------------------------|
| 「何を（What）提供するか」 | 「なぜ（Why）それを選ぶのか」 |
| 軽量、高性能、低価格 | ブランドの背景、想い、世界観 |
| スペック表での比較 | 比較不可能な独自の価値 |
| 記憶に残りにくい | 感情に響き、記憶に刻まれる |

### プロダクトライフサイクルと価値の移行
- 導入期・成長期 → **機能的価値の追求**（課題解決・スペック）
- 成熟期・飽和期 → **情緒的価値・自己表現価値の追求**（ブランドミーニング）

市場が成熟し機能差がなくなった時、バリューチェーン全体に「独自の意味」が宿っているかが問われる。

### ケーススタディ

**富士フィルムのV字回復**
- 技術資産（What）は変えずに、その意味（Why）を書き換えた
- 写真フィルムの技術（コラーゲン制御・抗酸化技術・ナノ化技術）
- 「写真を綺麗に残す」技術 → 「人間の肌（美しさ・生命）を保つ」技術へ
- → ASTALIFT（アスタリフト）としてヘルスケア領域へ進出
- 「フィルムの会社」→「美と健康を守る会社」へ存在意義を再定義

**Red Bullの圧倒的ポジション**
- 商品：エナジードリンク（カフェイン・糖分）
- 意味：挑戦、熱量、エクストリームスポーツ、カルチャー（翼をさずける）
- 競合の多い飲料市場で「美味しさ」や「価格」で勝負していない
- 消費者は成分を買うのではなく、「挑戦者への賛歌」というストーリーに共感して対価を払っている

### 顧客インサイトへの潜水：隠れた「Why」を探る
- **水面上のニーズ（Needs）**：「喉が渇いた」「疲れた」「便利なツールが欲しい」→ 競合も容易に把握でき、すぐに機能競争に陥る
- **水面下のインサイト（Insight）**：「限界を超えたい」「平凡な自分から抜け出したい」「本当はもっと認められたい」→ 顧客の心の奥底にある本音。ここにブランドの「意味」を接続する

### 模倣不可能なバリュープロポジションの設計
**顧客が強く望み、自社だけが提供できる「Must Have」な価値を定義する**
- 顧客が望む価値
- 自社が提供できる価値
- 競合他社が提供できない価値
→ 3つが重なる領域 = **バリュープロポジション**

### インサイト→コミュニケーションコンセプトへの変換
- 顧客インサイト・自社の想い・製品の機能（原石）
- → 意味の言語化・コンセプト設計（プリズム）
- → コミュニケーションコンセプト（すべてのクリエイティブの軸）

コンセプトが定まることで、ウェブサイト・広告クリエイティブ・SNS発信・パッケージデザインに至るすべての顧客接点に「一貫した意味」が宿る。

### 虎谷事例：DECORTE リップカラー大ヒット
- ラグジュアリーリップはDior・CHANEL・YSLで70%の超寡占市場
- 市場最大ボリュームの奪取は狙わず、**Dior/CHANEL/YSLの「+1」想起NO.1**を狙った
- 打ち出した独自機能：肌機能（スキンケア級）・日本人に最適な血色感シェード・日常落ちしないラスティング
- 提供価値：「（メイクなのに）ツヤ・血色感ある肌の色を実現する日常使いに最適なリップ」
- 社内でポジショニングについてめちゃくちゃ怒られたが、勝手に敢行
- **2週間で16万本出荷、空前の大ヒット**

### 明日からできる第一歩：コピーに「意味」を宿す
**機能訴求（スペック）**
> 「このスニーカーは超軽量素材を使用し、クッション性と通気性に優れています。」  
→ 反応：「ふーん、便利そうだね（でも他社のも軽いよね）」

**ストーリーテリング（意味）**
> 「足の痛みに悩むランナーの生の声から生まれました。何百人もの走りを観察し、軽さだけでなく『足への究極の優しさ』を追求した一足です。」  
→ 反応：「私の悩みを分かってくれている。このブランドを応援したい。」

**主語を「自社の技術」から「顧客の体験と想い」に変えるだけで、機能は「意味」に変わる。**

### 結論：sirokへの示唆
```
「最強のエンジン」（D2C運用力）＋「確かなコンパス」（ブランドの意味）= さらなる成長
```

5つのステップ：
1. **脅威の認識**：機能・広告量競争の限界を悟る
2. **パラダイムシフト**：「モノ（機能）」から「意味（世界観）」へ価値を移行させる
3. **インサイトの発見**：顧客の心の奥底にある「Why」を見つけ出す
4. **独自価値の設計**：競合が模倣できないバリュープロポジションを定義する
5. **一貫した実行**：すべてのコミュニケーションに「意味」を宿らせる

### COSME DECORTE × 大谷翔平キャンペーン（虎谷氏直接担当）の全貌
- Web広告投資額：**0円**（社内で322回却下されながら実行）
- 洞察：メディアの収益構造＝視聴率→広告費→制作費。大谷映像は海外素材購入コストが高くメディアの負担が大きい
- 戦略：契約（3年・40億円）を活かし「動く大谷」映像をメーカー側で制作→メディアに無償提供
- OOH展開：全国百貨店の壁面・懸垂幕をジャック。深夜施工→翌朝メディアが取材
- 結果：広告投資約1億円→広告換算額約7億円
- **落とし穴**：認知率約70%まで到達したが「話題で買う層」（本来ターゲット外）を大量獲得→リピート率低下・維持コスト急増。2024年に水原スキャンダルで中断

### 大谷起用から得た教訓：「200%成長の罠」
> 「200%成長はいらない、毎年持続的な100%を積み上げるべき」
- 話題が広がりすぎると、取れた客層の維持コストが急増する
- **「伝えるべき人に伝わること」が本来の話題化。「多くの人に知られること」とは別物**
- 200%成長した時は「届けたい人が買ってくれて伸びたのか？」の中身を必ず見る

### 虎谷氏のインサイト特定手法

**アンケート調査がインサイトに届かない理由**
- どのブランドも買っている人にはターゲット外のマジョリティ層が大量に含まれる
- アンケートでターゲットを抽出しようとすると、キャズム後のマジョリティ層の出現率が高くなる
- 結果：「何を参考にしますか？」→ 安いもの・口コミ・SNS しか出てこない（どのファネルでも同じ答え）
- **アンケート調査でインサイトに辿り着くことは100%不可能**

**有効な手法：ヒアリング → 仮説構築 → 小規模検証**
1. 10〜20人にヒアリング（百貨店で「すみません、時間いいですか」と直接聞くレベル）
2. そこから仮説を構築
3. ターゲットを絞って広げた中にコンセプトを突き合わせて検証

**インタビュー設計のコツ：「何を明らかにしたいか」を先に決める**
- 何でもかんでも聞かない。聞くことがブレると答えもブレる
- 例：「リップに求める機能は何ですか？」→ 当然ファンクショナルな答えに寄っていく（NG）
- 今回明らかにしたいのはインサイト → 聞き方を絞る

**DECORTEリップでの仮説構築プロセス（実例）**
- 起点となったストーリー：「Dior/CHANEL/YSLと戦わない、プラスワンポジションを取る」
- 仮説：リップって鮮やかに自分を彩りたいときと日常・仕事中で求める機能の差があるのでは？（着る洋服・メイクが違うのと同じ）
- 質問を絞る：「仕事中と土日で使っている色は違いますか？」だけ聞く
- 違う人にはさらに深掘り：「ブランドは何を分けてる？」「色味は？」

**仮説の作り方：自分がどうしたいかが起点**
- 「作りたいストーリー」と「独自のポジション」を先に決める
- その独自性から、構築すべき仮説が変わる → 聞くことが変わる
- AIを壁打ちパートナーとして活用することで仮説構築が加速

**思考の起点**
> 「マスに突っ込んでいくより独自のポジションを形成し、独自の価値を伝えることが完全な前提。だから私の背景は競争戦略一択。」

**思考の順序：ターゲットから入らない**

一般的な順序（NG）：
> ターゲット → インサイト → ニーズ → コンセプト

虎谷氏の順序：
> **コアバリュー** → インサイト → ニーズ → ターゲット

- 「こんなターゲットだからこんなインサイト」とは考えない
- 先に「このコアバリューだったら、このインサイトに刺さる」が起点
- そのインサイトを持っている人のニーズを考え、結果としてターゲット像が決まる
- 「誰に届けるか」ではなく「こういう気持ちを持っている人を解決できる」が先

**セグメントが小さすぎた場合の対処**
- 試算してみて「事業上小さすぎる」となってもOK
- その場合はコアバリューを分解して、もう一つ小さな塊（別セグメント）を作る
- **浅く広く普遍的なコモディティ化した価値は絶対に作らない**
- 絞って絞って「その人一人に」を積み上げていく

### カスタマージャーニー設計
- 3ヶ月サイクル：初動設計→1ヶ月で初速確認→残り2ヶ月で調整
- 「目立つ」はVMDだけでなく入口・接客・レジ・同梱等カスタマージャーニー全体の問題
- コミュニケーションコンセプトのブレは全チャネルに波及する広告費のロス

### N organicへのネクストアクション（講義内で合意）
- N organicの「選ばれる意味（インサイト）」を社内で議論・言語化する（MK部門・5月末）
- コミュニケーションコンセプトの有無を確認し、なければ策定着手（水谷・5月末）
