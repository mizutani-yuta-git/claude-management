# YouTubeスポンサーシップ メニュー調査

調査日：2026-04-03

## メニュー種類

| フォーマット | 内容 |
|---|---|
| **ミッドロール（中盤スポット）** | 動画中盤に30〜90秒のスポンサーコーナー。最も一般的・高単価 |
| **フルインテグレーション** | 動画企画に商品を自然に組み込む（コスメレビュー等）。ミッドロールより高め |
| **専用動画（Dedicated Video）** | 動画全体が1ブランドの紹介。最上位プラン |
| **プレロール（冒頭）** | 開始直後に30〜60秒。ミッドロールの70〜80%程度の価格 |
| **ポストロール（末尾）** | 最も安価。離脱率が高い |
| **タイアップ動画（コラボ動画）** | 商品を主役にしたオリジナル動画を制作。日本で最も主流 |
| **SNSセット** | YouTube + Instagram / TikTok を組み合わせたクロスプラットフォームパッケージ |
| **長期パートナーシップ** | 数ヶ月〜1年間の継続契約。ブランドアンバサダー的な起用 |

## 料金相場（登録者数別）

| 登録者規模 | タイアップ動画（1本） | 計算式の目安 |
|---|---|---|
| 〜5万人 | 10〜30万円 | 登録者 × 2〜5円 |
| 10万〜50万人 | 40〜200万円 | 登録者 × 4〜8円 |
| 50万〜100万人 | 150〜500万円 | 登録者 × 3〜5円 |
| 100万人超 | 300〜1,000万円以上 | 登録者 × 3〜5円 |

- 美容・コスメ系は「購買意欲の高い女性オーディエンス」として単価がやや割高
- 専用動画はミッドロールの2〜4倍（最大10倍）のコスト
- 二次利用（広告転用）は +20〜50% のライセンス料が発生

## 主要エージェンシー

| 会社名 | 特徴 |
|---|---|
| **UUUM** | 日本最大MCN。美容特化チーム「Beauty insight」あり |
| **VAZ** | Z世代・ビューティー案件に強み |
| **find model** | 8,000人以上のインフルエンサー登録。月額〜案件型 |
| **ユウメイキャスティング** | 美容系YouTuberキャスティング実績多数 |

エージェンシー仲介手数料はインフルエンサー報酬の10〜30%が相場。

## 注意点

- 料金は非公開が原則。メディアキットの公開は少なく、交渉はエージェンシー経由が一般的
- #PR / #広告の明示が義務（景表法）。コスメは薬機法の確認も必要
- インフルエンサーマーケティング市場は2025年に約1,021億円（前年比+19%増）と拡大中
